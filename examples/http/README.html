HTTP Integration Example¶

This example shows how to use WSGI Middleware and requests integrations to instrument an HTTP client and server in Python.

The source files required to run this example are available here.

Installation¶

pip install opentelemetry-api
pip install opentelemetry-sdk
pip install opentelemetry-ext-wsgi
pip install opentelemetry-ext-http-requests
pip install flask

Run the Example¶

  • Run the server

python server.py

  • Run the client from a different terminal

python client.py

The output will be displayed in the console on the client side:
